# finance-tracker-frontend

Frontend-часть финансового трекера на React + Vite.

## Стек

- React
- Vite
- JavaScript
- CSS
- Docker
- Nginx

## Что реализовано

- регистрация пользователя
- вход и выход из аккаунта
- удаление аккаунта
- добавление доходов и расходов
- выбор категории
- выбор валюты BYN / RUB / USD
- просмотр списка транзакций
- удаление транзакций
- просмотр баланса в выбранной валюте
- отображение ошибки при недостатке средств

## Запуск локально

```bash
npm install
npm run dev
